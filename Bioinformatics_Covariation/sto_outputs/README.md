# R-scape Covariation Analysis Pipeline — Stockholm file outputs

This repository contains the .sto file alignments for various riboswitches generated in this project.

---

## 💻 General Pipeline

1) The .arnold.sto was generated by combing aptamer alignments generated with nhmmer and terminator alignments generated with MUSCLE. 
2) [Rfam_ID]_[riboswitch].arnold.sto is the input for CaCoFold
3) [Rfam_ID]_[riboswitch].arnold.cacofold.sto is the output from CaCoFold
4) XRiboswitch covariation model is built with cmbuild
5) [Rfam_ID]_[riboswitch].arnold.cacofold.sto is calibrated with cmcalibrated
6) Calibrated [Rfam_ID]_[riboswitch].arnold.cacofold.sto motif is then searched with cmsearch of all the extended aptamer sequences
7) The output from cmsearch is then the input for CaCoFold
8) [Rfam_ID]_[riboswitch].all.sto is the output from CaCoFold after searching all the extended aptamer sequences

| File | Description |
|------|-------------|
| `Rfam_ID]_[riboswitch].arnold.sto` | Input to CaCoFold after running ARNold. |
| `[Rfam_ID]_[riboswitch].arnold.cacofold.sto` | Output from CaCoFold after running ARNold. |
| `[Rfam_ID]_[riboswitch].all.sto` | Output from CaCoFold after running cmsearch. |
| `XRiboswitch.cm` | Riboswitch covariation model |
| `[Rfam_ID]_NCBI_Extended.fa` | FASTA file output after extended the Rfam aptamers with the downstream genome sequence from the NCBI |

---

## 🧪 Terminal Commands for R-scape Analysis  
**Run date: 20 August 2024**

### 🔹 Run CaCoFold on ARNold alignment
```bash
R-scape --cacofold [Rfam_ID]_[riboswitch].arnold.sto
```
### Build the covariation model
```python
cmbuild XRiboswitch.cm [Rfam_ID]_[riboswitch].arnold.cacofold.sto
```
### Calibrate CaCoFold Output
```python
cmcalibrate XRiboswitch.cm
```
### 🔹 Search all extended aptamers for the covaraitaion model
```bash
cmsearch -A [Rfam_ID]_[riboswitch].all.sto XRiboswitch.cm [Rfam_ID]_NCBI_Extended.fa
```
### 🔹 CaCoFold on the output from cmsearch
```bash
R-scape --cacofold [Rfam_ID]_[riboswitch].all.sto
```
